格式說明

  • 使用 @proto.go、@define.go 文件
  • 說明：、HTTP Action：，後面的你不要自行增加內容；若欄位本身有值，你也不要自行增加內容
  • 當Response內容中，有 Data 時，API規格中需要新增 Data 表格中的內容
  • 「是否必填」欄位，若註解說明中未提及的話，都為「是」
  • 「驗證條件／限制」欄位，都保持空欄位；若欄位本身有值，你不要自行增加內容
  • 狀態碼內容請根據「define.go」檔案

各 API 文件

AuthAuthPreLogin - 驗證前置登入

說明：[驗證]前置登入

HTTP Action：/auth/pre-login

Request

參數名稱 屬性 是否必填 說明 驗證條件／限制
ClientID string 否 客戶端識別碼(留空則由後台自動生成指派,唯一碼UUID)

Response

參數名稱 屬性 是否必填 說明 驗證條件／限制
ErrResp ErrResp 是 錯誤回應
StatusCode AuthPreLoginStatus 是 前置登入狀態碼
ClientID string 是 客戶端識別碼(唯一碼UUID)
CaptchaID string 是 驗證碼編號(唯一碼UUID)
CaptchaImage string 是 驗證碼PNG圖片的二進制位元資料轉成Base64格式 像素：120x40，6位數
ClientIDExpiredTime int64 是 客戶端識別碼過期時間戳(UTC+0)
CaptchaIDExpiredTime int64 是 驗證碼過期時間戳(UTC+0)
狀態碼 狀態名稱 狀態訊息
0 AuthAuthPreLoginSuccess 前置登入成功
1 AuthAuthPreLoginClientIDNotExist 客戶端識別碼不存在(已過期)

說明：[驗證]刷新驗證碼

HTTP Action：/auth/refresh-captcha

Request

參數名稱 屬性 是否必填 說明 驗證條件／限制
ClientID string 是 客戶端識別碼(唯一碼UUID)
CaptchaID string 是 驗證碼編號(唯一碼UUID)

Response

參數名稱 屬性 是否必填 說明 驗證條件／限制
ErrResp ErrResp 是 錯誤回應
StatusCode AuthRefreshCaptchaStatus 是 刷新驗證碼狀態碼
CaptchaID string 是 驗證碼編號(唯一碼UUID)
CaptchaImage string 是 驗證碼圖片的二進制位元資料轉成Base64格式 像素：120x40，6位數
ClientIDExpiredTime int64 是 客戶端識別碼過期時間戳(UTC+0)
CaptchaIDExpiredTime int64 是 驗證碼過期時間戳(UTC+0)
狀態碼 狀態名稱 狀態訊息

AuthLogin - 驗證登入

說明：[驗證]登入

HTTP Action：/auth/login

Request

參數名稱 屬性 是否必填 說明 驗證條件／限制
ClientID string 是 客戶端識別碼
CaptchaID string 是 驗證碼編號(唯一碼UUID)
CaptchaAnswer string 是 驗證碼答案
AgentID string 是 登入代理帳號
AgentPswd string 是 登入代理密碼
ClientLoginIP string 是 客戶端登入IP

Response

參數名稱 屬性 是否必填 說明 驗證條件／限制
ErrResp ErrResp 是 錯誤回應
StatusCode AuthLoginStatus 是 登入狀態碼
AgentToken string 是 代理令牌 若使用者重複登入，更新Token，且只有新端點能操作，舊端點無法操作
AgentTokenExpiredTime int64 是 代理令牌過期時間戳(UTC+0)
AgentID string 是 代理帳號
AgentName string 是 代理姓名
AgentBalane string 是 代理餘額
狀態碼 狀態名稱 狀態訊息
0 AuthLoginSuccess 登入成功
1 AuthLoginClientIDNotExist 客戶端識別碼不存在(已過期)
2 AuthLoginCaptchaIDNotMatch 驗證碼編號不符
3 AuthLoginCaptchaIDExpired 驗證碼編號已過期
4 AuthLoginCaptchaError 驗證碼錯誤
5 AuthLoginAgentIDNotExist 代理帳號不存在
6 AuthLoginAgentPswdError 代理密碼錯誤
7 AuthLoginAgentActiveDisabled 代理帳戶已被禁用

AgentCreate - 代理新增代理

說明：[代理]新增代理

HTTP Action：/agent/create

Request

參數名稱 屬性 是否必填 說明 驗證條件／限制
AgentToken string 是 代理令牌
CreateAgentID string 是 新增代理帳號
CreateAgentName string 是 新增代理姓名
CreateAgentPswd string 是 新增代理密碼
CreateAgentEmail string 否 新增代理郵箱(選填)
CreateAgentPhone string 否 新增代理手機號(選填)
CreateAgentProfitRate string 是 新增代理佔成比例(%) 預設為 0；不得超過上層剩餘比例；區間 0.00 ~ 100.00；有效小數點：2 位
CreateAgentPrevGameCommiRate string 是 新增代理賽事前返佣比例(%) 預設為 0；間隔調整為0.05%；最高上限為2%；可超過上層剩餘返佣比例；有效小數點：2 位
CreateAgentInGameCommiRate string 是 新增代理賽事中返佣比例(%) 預設為 0；間隔調整為0.05%；最高上限為2%；可超過上層剩餘返佣比例；有效小數點：2 位

Response

參數名稱 屬性 是否必填 說明 驗證條件／限制
ErrResp ErrResp 是 錯誤回應
StatusCode AgentCreateStatus 是 新增代理狀態碼
CreateAgent AgentData 是 新增代理資料
狀態碼 狀態名稱 狀態訊息
0 AgentCreateSuccess 新增代理成功
1 AgentCreateAgentTokenInvalid 代理令牌無效(已過期)
2 AgentCreateAgentNotPermission 代理無管理權限
3 AgentCreateAgentIDFormatError 新增代理帳號格式錯誤
4 AgentCreateAgentIDIsExist 新增代理帳號已存在
5 AgentCreateAgentNameFormatError 新增代理姓名格式錯誤
6 AgentCreateAgentNameIsExist 新增代理姓名已存在
7 AgentCreateAgentPswdEmpty 新增代理密碼為空
8 AgentCreateAgentPswdFormatError 新增代理密碼格式錯誤
9 AgentCreateProfitRateInvalid 新增代理佔成比例無效
10 AgentCreatePrevGameCommiRateInvalid 新增代理賽事前返佣比例無效
11 AgentCreateInGameCommiRateInvalid 新增代理賽事中返佣比例無效
12 AgentCreateFailed 新增代理失敗

FixActiveStatus string 否 修改帳戶啟用狀態[若此欄為空,則表示不修改; "0":停用, "1":啟用]

Response

參數名稱 屬性 是否必填 說明 驗證條件／限制

StatusCode AgentUpdateAccountStatus 是 更新代理帳戶狀態碼
UpdateAgent AgentData 是 更新代理資料

0 AgentUpdateAccountSuccess 更新代理帳戶成功
1 AgentUpdateAccountAgentTokenInvalid 代理令牌無效(已過期)
2 AgentUpdateAccountUpdAgentIDNotExist 更新代理帳號不存在
3 AgentUpdateAccountUpdAgentNotPermission 更新下層代理無管理權限
4 AgentUpdateAccountFixAgentNameFormatError 修改代理姓名格式錯誤
5 AgentUpdateAccountFixAgentNameIsExist 修改代理姓名已存在
6 AgentUpdateAccountFixAgentPswdFormatError 修改代理密碼格式錯誤
7 AgentUpdateAccountFixActiveStatusFormatError 修改代理啟用狀態格式錯誤

AgentQueryDownAgentList - 代理查詢下層代理列表

說明：[代理]查詢下層代理列表

HTTP Action：

Request

參數名稱 屬性 是否必填 說明 驗證條件／限制
AgentToken string 是 代理令牌
QueryAgentID string 否 查詢代理帳號(若此欄為空,則表示查詢自身下層代理列表, 若此欄有值,則查詢所有下層樹的某個指定代理的下層代理列表)
ListIndex uint32 是 查詢下層代理列表索引(指定頁數,從0開始)
ListCount uint32 是 查詢下層代理列表數量(每頁數量上限,0:默認25筆資料)
MatchByDownAgentID string 否 符合指定條件的下層代理帳號(支持萬用字元:%) 使用 SQL LIKE 語法：模糊查詢
MatchByDownAgentName string 否 符合指定條件的下層代理姓名(支持萬用字元:%) 使用 SQL LIKE 語法：模糊查詢
MatchByDownAgentField map[string]string 是 符合指定下層代理欄位條件 map[AgentField, Value](舉例: ["ActiveStatus","1"])

Response

參數名稱 屬性 是否必填 說明 驗證條件／限制
ErrResp ErrResp 是 錯誤回應
StatusCode AgentQueryDownAgentListStatus 是 查詢下層代理列表狀態碼
QueryAgentTree AgentTree 是 當前代理樹結構
DownAgentList QueryDownAgentData 是 下層代理資料列表
狀態碼 狀態名稱 狀態訊息
0 AgentQueryDownAgentListSuccess 查詢下層代理列表成功
1 AgentQueryDownAgentListAgentTokenInvalid 代理令牌無效(已過期)
2 AgentQueryDownAgentListQueryAgentIDNotExist 查詢下層代理帳號不存在
3 AgentQueryDownAgentListQueryAgentNotPermission 查詢下層代理無管理權限
