功能操作 權限階層限制 說明 限制條件
查詢 Root、SH、Agent 階層 根據以下篩選條件查詢資料

條件：下拉式選單（會員帳號、會員姓名），在輸入框中，使用模糊查詢特定帳號資料（預設：空）

帳號啟用狀態：下拉式選單（全部、啟用、停用），用於篩選資料（預設：全部）

每次查詢為25筆資料，剩餘資料透過頁碼查詢		 Root、SH 階層，顯示所有會員資料於列表中

Agent階層，只會看到自己下層的直屬會員
，以下為例：
- Agent 1（Member 1, Member 2）無法看到 Agent 2（Member 3, Member 4）底下的資料
重置 Root、SH、Agent 階層 將篩選條件變更為預設值
編輯會員 Root、SH、Agent 階層 點擊後跳轉至「編輯會員」頁面 需要點擊列表中任一代理資料，按鈕才會顯示
會員充提 Agent 階層 點擊後跳轉至「會員充提」頁面 需要點擊列表中任一代理資料，按鈕才會顯示
當前層級 Root、SH、Agent 階層 顯示當前所查詢的層級帳號 只顯示 當前帳號以下的層級，以下為例：

情境一
AgentID = Root；AgentRank = Root；前端的「當前層級」顯示方式為「當前層級：」
點擊「上層代理帳號」帳號後，前端「當前層級」顯示方式為「當前層級：上層代理帳號」

情境二
AgentID = Agent；AgentRank = Agent；前端的「當前層級」顯示方式為「當前層級：」
由於此層級沒有下一層所以無法點擊「上層代理帳號」
停權代理 / 解除停權 除 Agent 層級外的所有層級 點擊「停權」或「解除停權」按鈕

「啟用」變更為「停權」時，帳號禁止進入前台，且會回收該帳號之帳戶餘額

「啟用」變更為「停權」時恢復狀態為「正常」，允許帳號進入前台		 「啟用」變更為「停權」，跳出提示訊息
{{
確定要停權此會員帳號嗎？
會員帳號：xxx
帳戶餘額：xxx
}}
點擊「確認」後，啟用狀態才生效

「啟用」變更為「停權」，跳出提示訊息
{{
確定要啟用此會員帳號嗎？
會員帳號：xxx
}}
點擊「確認」後，啟用狀態才生效

在停用會員帳號時，會進行充值，系統在備注欄位寫上「分數回收」

欄位說明

DB

欄位名稱 說明 驗證條件／限制
會員帳號 會員登入使用的帳號，系統唯一識別標籤 必填；英文或數字組成；至少 4 碼；最多 20 碼；不可與現有會員帳號重複
會員密碼 登入用的密碼，建立後須經加密儲存 必填；至少 4 碼；最多 20 碼；允許特殊字元（例：`! @ # $ % ^ & * ( ) _ + - = [ ] { }）
會員姓名 會員真實姓名或暱稱，用於識別身份 必填；中英文不限；最多 20 碼
上層代理帳號 會員所屬代理的帳號，自動綁定 系統自動填入；由當前登入的 Agent 建立者自動設定
狀態 控制該會員帳號是否可登入 必填；可選「正常」或「停權」「停權」將無法登入前台
帳戶餘額 會員目前的錢包可用餘額 必填；預設為 0；若填入金額，需檢查建立者帳號餘額是否足夠轉入
當前代理賽事前返佣比例（%） 代理可使用的賽事前返佣比例 系統自動填入，不可修改
當前代理賽事中返佣比例（%） 代理可使用的賽事中返佣比例 系統自動填入，不可修改
賽事前下層返佣比例（%） 依據賽事開始前之抽水，從代理獲得的返佣比例，再將比例分給底下會員 必填；預設為 0；間隔調整為0.05%；最高上限為2%；可超過上層剩餘返佣比例；有效小數點：2 位
賽事中下層返佣比例（%） 依據賽事進行中之抽水，從代理獲得的返佣比例，再將比例分給底下會員 必填；預設為 0；間隔調整為0.05%；最高上限為2%；可超過上層剩餘返佣比例；有效小數點：2 位
電子郵件 聯絡與通知用途 選填
手機號碼 聯絡用手機門號 選填
創立時間 該帳號建立的時間戳記 系統自動填入，不可修改
更新時間 最近一次修改會員資料的時間 系統自動更新，不可手動編輯

功能操作說明

功能操作 權限階層限制 操作方式 系統行為說明
查詢會員資料 所有代理層級 1.根據「條件」輸入關鍵字後，點擊「查詢」按鈕
2.直接點擊「篩選條件」
3.點擊特定頁碼
 顯示系統內所有會員資料，每頁顯示 25 筆資料
根據「條件」的選單內容：會員帳號、會員姓名，模糊查詢
若有多個「篩選條件」，採用 AND 邏輯來過濾資料
未設定條件時，預設顯示最近創建時間的資料
新增會員 僅限 Agent 層級 填寫必填欄位（會員帳號、會員姓名）後提交 建立會員帳號，寫入基本資料並初始化餘額
編輯會員 所有代理層級 修改會員欄位內容後點擊「儲存」 更新會員名稱、返佣比例、電子郵件、手機號碼等可編輯欄位資料。返佣比例於維護時間區間內即時生效。
會員充提 僅限 Agent 層級（不可跨層充提） 選擇交易類型與交易金額後，點擊「確認交易」 對會員進行上下分之功能
重置會員密碼 所有代理層級 點擊「重置密碼」按鈕 將該會員密碼重置為預設值 0000，系統提示成功訊息
停權會員 / 解除停權 所有代理層級 點擊「停權」或「解除停權」按鈕，並進行二次確認 停權：變更會員狀態為「停權」，禁止登入前台，且會回收該帳號之帳戶餘額
解除：恢復狀態為「正常」，允許登入

使用之API

  • 會員：查詢會員
    • 點擊導航區「會員列表」時
    • 返回「會員列表」頁面時
    • 「會員列表」頁面重新整理時
    • 使用條件查詢
  • 會員：新增會員
    • 點擊「新增會員」按鈕時
  • 會員：取單一會員資料
    • 打開「編輯會員」頁面時
    • 「編輯會員」頁面重新整理時
  • 會員：編輯會員
    • 在「編輯會員」頁面時，點擊「確認編輯」按鈕時
  • 會員：重置會員密碼
    • 在「編輯會員」頁面時，點擊「重置會員」按鈕時
  • 會員：變更帳號啟用狀態
    • 在「會員列表」的「操作」欄位中，點擊「帳號啟用狀態」之 icon
  • 金流：取單一會員帳戶餘額
    • 打開「會員充提」頁面時
    • 「會員充提」頁面重新整理
    • 在「會員充提」頁面，點擊「重置」按鈕時
  • 金流：取單一當前帳戶餘額
    • 打開「會員充提」頁面時
    • 「會員充提」頁面重新整理
    • 在「會員充提」頁面，點擊「重置」按鈕時
  • 金流：會員上下分
    • 在「會員充提」頁面，點擊「確認交易」按鈕時
