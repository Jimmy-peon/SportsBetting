代理列表

功能操作 權限階層限制 說明 限制條件
查詢 除 Agent 層級外的所有層級 根據以下篩選條件查詢資料

條件：下拉式選單（代理帳號、代理姓名），在輸入框中模糊查詢特定帳號資料（預設：空）

帳號啟用狀態：下拉式選單（全部、啟用、停用），用於篩選資料（預設：全部）

每次查詢為25筆資料，剩餘資料透過頁碼查詢		 各層級層可以查閱到的層級資料，以下為例
- 層級 Root ，可以查閱 SH ~ Agent 階層的資料
- 層級 SH，可以查閱 SSMA ~ Agent 階層的資料
- 層級 SSMA，可以查閱 SMA ~ Agent 階層的資料
- 層級 SMA，可以查閱 MA ~ Agent 階層的資料
- 層級 MA，可以查閱 Agent 階層的資料

不使用「條件」功能的話，列表只會顯示當前查詢的代理帳號下的直屬代理，以下為例
- 層級 SSMA，只可以查閱 SMA 階層的代理資料

當層代理無法查閱上層代理資料，以下為例：
- 層級 SMA ，看不到所有 SSAM 代理資料

無法查閱不同支線下的資料，以下為例：
-SSMA 1（SMA 1, SMA 2）無法看到 SSMA 2（SMA 3, SMA 4）底下的資料
重置 除 Agent 層級外的所有層級 將篩選條件變更為預設值
編輯代理 除 Agent 層級外的所有層級 點擊後跳轉至「編輯代理」頁面 需要點擊列表中任一代理資料，按鈕才會顯示
代理充提 除 Agent 層級外的所有層級 點擊後跳轉至「代理充提」頁面 需要點擊列表中任一代理資料，按鈕才會顯示
當前層級 除 Agent 層級外的所有層級 顯示當前所查詢的層級帳號 AgentTree (QueryAgentTree) 呈現方式，此文件中檔案區「層級查詢」方式

只顯示 當前帳號 AgentRank 以下的層級，以下為例
情境一（如影片）
AgentID = SH；AgentRank = SH；前端「當前層級」的顯示方式為「當前層級：SH」
點擊查詢 SSMA 1 時，前端顯示「當前層級」方式為「當前層級：SH 1 > SSMA 1」
再點擊查詢 SMA 1 時，前端「當前層級」的顯示方式為「當前層級：SH 1 > SSMA 1 > SMA 1」

情境二
AgentID = SMA 1；AgentRank = SMA；前端「當前層級」的顯示方式為「當前層級：SMA 1」
點擊查詢 MA 1 時，前端「當前層級」的顯示方式為「當前層級：SMA 1 > MA 1」
停權代理 / 解除停權 除 Agent 層級外的所有層級 點擊「停權」或「解除停權」按鈕

「啟用」變更為「停權」時，帳號禁止進入後台，且會回收該帳號之帳戶餘額

「啟用」變更為「停權」時恢復狀態為「正常」，允許帳號進入		 「啟用」變更為「停權」，跳出提示訊息
{{
確定要停權此代理帳號嗎？
代理帳號：xxx
帳戶餘額：xxx
}}
點擊「確認」後，啟用狀態才生效

「啟用」變更為「停權」，跳出提示訊息
{{
確定要啟用此代理帳號嗎？
代理帳號：xxx
}}
點擊「確認」後，啟用狀態才生效

在停用代理帳號時，會進行充值，系統在備注欄位寫上「分數回收」

欄位說明

欄位名稱 說明 驗證條件／限制
代理帳號 欲建立的代理登入帳號，需具備唯一性 唯讀狀態，系統自動填入，不可修改
代理姓名 代理真實姓名或暱稱，用於識別身份 唯讀狀態，系統自動填入，不可修改
上層代理帳號 自動帶入目前登入者帳號 唯讀狀態，系統自動填入，不可修改
帳號啟用狀態 該帳號建立後的啟用狀態 唯讀狀態，系統自動填入，不可修改
代理佔成比例（%） 下層代理佔成比例，為代理可獲得的投注分潤比例 唯讀狀態，系統自動填入，不可修改
賽前返佣比例（%） 下層代理可使用的賽事前返佣比例 唯讀狀態，系統自動填入，不可修改
走地返佣比例（%） 下層代理可使用的賽事中返佣比例 唯讀狀態，系統自動填入，不可修改
帳戶餘額 建立帳號時預設可用金額 唯讀狀態，系統自動填入，不可修改
操作 下拉式選單，對該代理進行「編輯代理」、「代理充提」、「停權帳號」等操作 點擊「編輯代理」、「代理充提」時，跳轉至對應頁面

重置 除 Agent 層級外的所有層級 將選填欄位變更回預設值

DB 欄位

欄位名稱 說明 驗證條件／限制
上層代理帳號 自動帶入目前登入者帳號 系統自動填入，不可修改
代理帳號 欲建立的代理登入帳號，需具備唯一性 必填；英文或數字組成；至少 4 碼；最多 20 碼；不可與現有會員帳號重複
代理密碼 該帳號登入所使用之密碼 必填；至少 4 碼；最多 20 碼；
允許特殊字元：`! @ # $ % ^ & * ( ) _ + - = [ ] { }
代理姓名 代理真實姓名或暱稱，用於識別身份 必填；中英文不限；最多 20 碼；
所屬層級 建立後自動設定為建立者的下一層級（不可手動選擇） 系統依據目前登入帳號層級自動判斷，限制不能越級建立
當前代理佔成比例（%） 代理可獲得的投注分潤比例 系統自動填入，不可修改；為上層代理所設定之比例
下層佔成比例（%） 當前代理階層將可獲得的投注分潤比例，分給下層代理 必填；預設為 0；不得超過上層剩餘比例；數字輸入；有效小數點：2 位
當前代理賽事前返佣比例（%） 代理可使用的賽事前返佣比例 系統自動填入，不可修改
當前代理賽事中返佣比例（%） 代理可使用的賽事中返佣比例 系統自動填入，不可修改
賽事前下層返佣比例（%） 依據賽事開始前之抽水，從代理獲得的返佣比例，再將比例分給底下代理 必填；預設為 0；間隔調整為0.05%；最高上限為2%；可超過上層剩餘返佣比例；有效小數點：2 位
賽事中下層返佣比例（%） 依據賽事進行中之抽水，從代理獲得的返佣比例，再將比例分給底下代理 必填；預設為 0；間隔調整為0.05%；最高上限為2%；可超過上層剩餘返佣比例；有效小數點：2 位
帳號啟用狀態 該帳號建立後的啟用狀態 必填；可選「正常」或「停權」若為「停權」則該帳號將無法登入後台
電子郵件 該代理聯絡用信箱 選填
手機號碼 聯絡用手機門號 選填
帳戶餘額 建立帳號時預設可用金額 必填；預設為 0；會出現負數金額；若填入金額；有效小數點：4 位
