運彩：後台球版功能：賽事管理

功能說明：管理所有前台可提供投注的足球比賽資料。支援賽事的新增、編輯、查詢與狀態控制。管理員可調整賠率、設定讓球與大小球盤口，並在賽事結束後執行賽果結算與投注派彩作業。亦可控制每場賽事是否開放於前台顯示與投注。

欄位說明

欄位名稱 說明 驗證條件／限制
賽事編號 系統內用以唯一識別每場比賽之識別碼，亦為查詢與對帳之主鍵欄位 必填；唯一值；格式固定為 [聯賽代碼]-[YYYYMMDD]-[流水號]；聯賽代碼，參考運彩：後台球版功能：聯賽對照表；流水號3碼
所屬聯賽 足球賽事所屬聯賽 必填；參考運彩：後台球版功能：聯賽對照表
主隊名稱 比賽主隊球隊名稱 必填；最多 30 個字元
客隊名稱 比賽客隊球隊名稱 必填；最多 30 個字元
賽事開始時間 賽事開始時間 必填；不可設置過去時間；格式：YYYY-MM-DD HH:MM
賽事狀態 控制賽事狀態
 必填；
「未開始」：系統自動填入，前台無法投注，開盤時間 < 當前時間
「開放投注」：系統自動填入，前台可以投注，開盤時間 > 當前時間
「暫停投注」：此狀態時，前台無法投注。可修改回「開放投注」狀態
「已結算」：由後台操作，前台無法投注。變更為狀態後，無法再變更成其他狀態
「取消」：由後台操作，前台無法投注。變更為狀態後，無法再變更成其他狀態
讓球隊伍 在前台顯示讓球隊伍 必填；需要為主隊名稱或客隊名稱
讓球數 讓球隊伍之讓球數 必填；預設為0；整數值
盤口讓球賠率 當賽事結束時，兩隊伍比分等於「讓球數」時，使用此賠率 必填；預設為0；有效小數點：0；有正負值；不超過100
讓球賽事前主隊賠率 賽事開始時間前使用的賠率 必填；預設為「返獎設定」頁面所設定值，可修改，設定區間為 0 ~ 2；有效小數位：2
讓球賽事前客隊賠率 賽事開始時間前使用的賠率 必填；預設為「返獎設定」頁面所設定值，可修改，設定區間為 0 ~ 2；有效小數位：2
讓球賽事中主隊賠率 賽事開始時間後使用的賠率 必填；預設為「返獎設定」頁面所設定值，可修改，設定區間為 0 ~ 2；有效小數位：2
讓球賽事中客隊賠率 賽事開始時間後使用的賠率 必填；預設為「返獎設定」頁面所設定值，可修改，設定區間為 0 ~ 2；有效小數位：2
大小球球數 整場賽事總進球數 必填；預設為0；整數值
盤口大小球賠率 當賽事結束時，總比分等於「大小球球數」時，使用此賠率 必填；預設為0；有效小數點：0；有正負值；不超過100
賽事前大球賠率 賽事開始時間前大球使用的賠率 必填；預設為「返獎設定」頁面所設定值，可修改，設定區間為 0 ~ 2；有效小數位：2
賽事前小球賠率 賽事開始時間前小球使用的賠率 必填；預設為「返獎設定」頁面所設定值，可修改，設定區間為 0 ~ 2；有效小數位：2
賽事中大球賠率 賽事開始時間後大球使用的賠率 必填；預設為「返獎設定」頁面所設定值，可修改，設定區間為 0 ~ 2；有效小數位：2
賽事中小球賠率 賽事開始時間後小球使用的賠率 必填；預設為「返獎設定」頁面所設定值，可修改，設定區間為 0 ~ 2；有效小數位：2
主隊最終比分 賽事結束後主隊最終分數 必填；預設為0；整數值
客隊最終比分 賽事結束後客隊最終分數 必填；預設為0；整數值
最終總進球數 賽事結束後主隊、客隊最終比分總和 必填；預設為0；整數值
結算時間 賽事結束時間 系統自動計算；不可修改
投注筆數 該賽事的投注總筆數 系統自動計算；不可修改；整數值
投注金額 該賽事的投注總金額 系統自動計算；不可修改；有效小數位：4
取消賽事原因 說明取消賽事的原因 最多 100 字
取消時間 賽事被取消時間 系統自動填入；不可修改
更新時間 資料最後一次更新的時間 系統自動填入；不可修改
創建時間 該賽事建立的時間戳記 系統自動填入，不可修改
創建方式 手動、自動 若透過「新增賽事」頁面，新增時為手動
若透過「爬蟲新增賽事」，新增時為自動
創建帳號 若創建方式為手動，顯示創建此賽事之帳號
若創建方式為自動，顯示Auto		 系統自動填入；不可修改

功能操作說明

功能操作 權限階層限制 操作方式 系統行為說明
查詢賽事 所有代理層級 進入賽事列表畫面 預設顯示當日創建時間最新的前 25 筆賽事資料，支援依賽事編號、聯賽等條件查詢
新增賽事 所有代理層級 點擊「新增賽事」並填寫表單 手動建立新賽事，系統標記來源為「手動建立」，必須填寫所有必填欄位後方可提交
編輯賽事 所有代理層級 點擊指定賽事後進入編輯畫面 可編輯賠率、讓球、盤口、狀態等欄位內容，部分欄位依狀態條件限制是否可修改。結算後，賽事所有欄位都不可修改
暫停/開放投注 所有代理層級 選擇目標狀態並點擊「修改狀態」 將賽事狀態變更為「開放投注」或「暫停投注」，影響前台顯示與是否可下單，變更即時生效
結算賽事 所有代理層級 點擊「結算賽事」並輸入最終比分 系統依照實際比分與盤口設定，自動計算每筆投注結果，更新會員帳戶餘額並鎖定賽事資料。
取消賽事 所有代理層級 點擊「取消賽事」後確認操作 將賽事狀態設為「取消」，系統自動處理所有已投注訂單之退款，並記錄取消原因
查看單一賽事 所有代理層級 點擊「查看詳情」後確認操作 當賽事狀態為已結算、取消後，賽事所有欄位都不可修改，僅供查看使用。
